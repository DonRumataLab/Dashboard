# М-Ризен: портал дашбордов

MVP внутреннего портала для контроля производственного и логистического плана.

## Что уже есть

- визуальный портал с разделами: обзор, производство, логистика, история, сравнение, загрузка;
- загрузка выгрузок 1С в `.xlsx` / `.xls`;
- отдельный дашборд контрольных дат производства импорт;
- хранение исторических снимков по выбранной дате в `localStorage`;
- выбор даты снимка для отображения дашбордов;
- сравнение двух загруженных дат;
- пустой стартовый портал, который наполняется только загруженными файлами.

## Ожидаемые колонки XLSX

### Контрольные даты импорт

Файл определяется автоматически, если в первой ячейке есть заголовок `контрольные даты`. Поддерживается структура, где:

- первая строка данных содержит коллекцию или сезон;
- вторая строка содержит названия этапов;
- значения в строках ниже являются датами этапов;
- дополнительные колонки с количеством дней между этапами допускаются и не мешают импорту.

### План-факт

Парсер понимает несколько вариантов русских названий колонок:

- заказ / номер заказа / заказ покупателя;
- артикул / номенклатура / код;
- изделие / товар / наименование / модель;
- цех / участок / подразделение;
- план / план шт / плановое количество;
- факт / факт шт / выпущено / готово;
- отгружено / отгрузка;
- брак / дефект;
- статус / состояние / этап;
- дата плана / плановая дата / срок / дата отгрузки.

## Запуск

```bash
npm install
npm run dev -- --port 5173
```

Портал будет доступен на `http://127.0.0.1:5173`.
